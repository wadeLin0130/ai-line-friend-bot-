# Security Policy

## Supported Versions

We currently support the latest version of the main branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing the maintainer or opening a private security advisory on GitHub (preferred).

Do **not** open a public issue for security problems.

We will respond within 48 hours and work with you to resolve the issue.

Thank you for helping keep this project secure!
